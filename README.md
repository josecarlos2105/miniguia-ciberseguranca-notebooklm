## 🔐 Resumo Estruturado – Cibersegurança

Cibersegurança é o conjunto de práticas externas para proteger sistemas, redes e dados contra ataques digitais.

---

## 🌐 Nmap

Ferramenta utilizada para:

- Descoberta de hosts  
- Varredura de portas  
- Identificação de serviços  

### 💻 Exemplo:
```bash
nmap -sS -p 80,443 192.168.0.1
🧱 Firewall (Linux)

Responsável por controlar o tráfego de rede.

Exemplo com iptables:

iptables -A INPUT -p tcp --dport 22 -j ACCEPT
📘 Glossário
Porta: canal de comunicação de rede
Firewall: sistema de filtragem de tráfego
Vulnerabilidade: falha explorável
Scan: processo de varredura
